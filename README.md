# Tic-Tac-Toe (C++)

A console-based Tic-Tac-Toe implementation written in C++, designed to demonstrate strong fundamentals in programming, problem-solving, and system design.

This project focuses on building a clean and maintainable codebase while progressively evolving the implementation using more advanced techniques and design patterns.

## Overview

The goal of this project is not just to recreate a simple game, but to use it as a foundation for practicing core software engineering concepts such as modular design, code organization, and incremental development.

The project will continuously evolve, with planned improvements in both architecture and functionality.

## Current Features

- Console-based user interface
- 3x3 game board
- Board initialization with indexed positions
- Structured output formatting

## Project Structure

tic-tac-toe-cpp/

## Build and Run

Compile the project using a C++ compiler:
g++ main.cpp -o game
./game


## Design Approach

The project is being developed in incremental steps, starting with a procedural approach to establish a solid understanding of game logic and control flow.

Future iterations will focus on:

- Refactoring into Object-Oriented Design
- Improving separation of concerns
- Enhancing scalability and maintainability

## Future Improvements

- Player input handling and validation
- Game state management
- Win and draw detection
- AI opponent (Minimax algorithm)
- Refactoring using Object-Oriented Programming
- Improved user interface
